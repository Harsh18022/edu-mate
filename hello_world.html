Hello World

  •                                 Hello World Program in C Language


                                The Hello World Program in C is the first program, just like in any other programming language, which is
                                 created to check if the basic setup and installation completed successfully or not.
                                  It is one of the simplest program you will learn.
                                
                                Below is a simple program printing Hello World in C language.
                                
                                
                                
                                    
                                //this is a comment
                                
                                #include     //including header file in our program
                                
                                int main()            //main() where the execution begins
                                {
                                    printf("Hello World");
                                    return 0;
                                }
                                
                                
                             
     
     Output  -  Hello World
