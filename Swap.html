                                Program of Swapping two numbers in C Language

                                Swapping of two numbers in C Language is the process in which the value of two variables is exchanged using some code.
                                
                                 For example:-
                                
                                a = 5, b = 4
                                // After swapping:
                                a = 4, b = 5
                                

                                
    
                                
                                         
                                        #include
                                            #include
                                            
                                            void main()
                                            {
                                                int x = 10, y = 15, temp;
                                                temp = x;
                                                x = y;
                                                y = temp;
                                                printf("x = %d and y = %d", x, y);
                                                getch();
                                            }
                                        
                                  
                                    
    
                                
                                
                             
     
                                Output:

x = 15  and  y = 10
